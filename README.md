# Gateball-IoT-Project
ゲートボールの得点を計算するためのアプリケーションのプロジェクトです。「For Watcher」の場合、黒い四角にマウスをクリックすることで得点が入ったことを判断させることが出来ます。「For Referee」の場合、1Pチームの場合「1」、「2」、「3」、「4」、「5」キーで赤チームの得点を、「6」、「7」、「8」、「9」、「0」キーで、青チームの得点を加算させることが出来ます。
